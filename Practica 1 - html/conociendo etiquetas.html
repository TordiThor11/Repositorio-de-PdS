Etiqueta Descripcion
Head La cabecera es una colección de información acerca del documento.
Base Proporciona una dirección de base para interpretar los enlaces relativos cuando el documento se lee fuera de su contexto (por ejemplo cuando está guardado en un disco).
Meta Proporcionar información sobre el documento que no pueda ser expresada en los campos anteriores. La información se escribe usando pares nombre/valor que pueden ser empleados para dos propósitos:
Meta 1er proposito Proporcionar datos al servidor de HTTP para que genere campos de cabecera (como por ejemplo la fecha de caducidad de un documento que se actualiza periódicamente). Ejemplo: <META HTTP-EQUIV ="Expires" CONTENT="Dec 1996">
Meta 2do proposito Para que un usuario incluya información adicional sobre el documento, como palabras clave o el nombre del autor. Ejemplo: <META NAME="Autor" CONTENT="TU NOMBRE">
BLOCKQUOTE Indica que un texto es una cita de otra fuente.
BR Indica un salto de linea
